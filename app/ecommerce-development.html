Case study

Waggle is the online store with the headquarters in Australia, they provide variety of dog gear. The owners looked for fully customized solution, so they decided to built the online store from scratch with us without using any ready-made engine.

Our role

We are the tech department for Waggle. In this case the client provides interface design, flows and graphics to be implemented.

Since 2011 we have been responsible for the backend development and frontend implementation of the web shop. Waggle owners since the beginning wanted to have a totally custom solution, so we designed software architecture, implemented the solution, optimized it with the growing traffic.

Currently we are still iterating, introducing new functionalities and improving internals of the system. The ecommerce is developing all the time so the story will be continued.

History

  • 2015

    To improve speed and scalability of the system we switched the hosting provider to AWS stack. Mobile version of the website was improved.

  • 2014

    A series of A/B tests of strategic pages was performed to improve the conversion rate. The checkout process got simplified.

  • 2013

    To improve the speed of the website with the growing userbase, content compression and caching were introduced. Business reports have been added to allow the owners easily see what's going on with the sales, margins, conversion rates, etc.

  • 2012

    We migrated all the existing data from the version of Waggle ecommerce that they used before 2011. The new layout of the website was implemented, PayPal Express integration completed, and the gift vouchers feature added. We moved the static assets to a CDN server.

  • 2011

    We discussed the possible technologies and proposed the best for the particular needs of the client. We designed software architecture and implemented the first version of Waggle.

Technology

The website has been created in a modular fashion so the whole system is loosely coupled and allows for adding new features easily. The website is integrated with a set of 3rd party services like payment providers, bookkeeping / repository trackers, and business analytics collectors.

Automatic tests have been prepared to make sure the software is stable regardless of the changes that are being introduced. A special care was taken when optimizing the speed of the website. We performed a series of speed tests and selected solutions that are optimal for the website and the current needs of the business.
